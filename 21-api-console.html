POST /v1/analysis/documents

Analyze uploaded documents using multi-agent AI. Returns structured findings, risks, and cross-document insights. Supports PDF, DOCX, XLSX, and PPTX formats.

Request Example 
curl -X POST https://api.dealscope.ai/v1/analysis/documents \
  -H "Authorization: Bearer ds_live_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "deal_id": "deal_abc123",
    "document_ids": ["doc_xyz789", "doc_def456"],
    "analysis_types": ["risks", "financials", "contracts"],
    "models": {
      "primary": "claude-opus",
      "validators": ["gpt-4-turbo", "gemini-pro"]
    },
    "options": {
      "cross_reference": true,
      "confidence_threshold": 0.85,
      "include_reasoning": true
    }
  }'
Parameters
Name Type Description
deal_id required string The unique identifier for the deal
document_ids required array List of document IDs to analyze
analysis_types array Types: risks, financials, contracts, entities
models.primary string Primary model: claude-opus, gpt-4-turbo, gemini-pro
models.validators array Models for cross-validation consensus
options.include_reasoning boolean Include step-by-step reasoning chains
